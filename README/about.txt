UI файлы:  # Предназначены для показанием окон и взаимодействием в них с проектом
1) main.ui
   1) Главное окно с показам данных
   2) Кнопки отвечающие за вызов инных окон или же редакции, загрузки или сохранения данных
2) products.ui
   1) Окно вызываемое кнопками редактирования и добавления
   2) Окно для редакции и добавление покупок
3) products_bd.ui
   1) Окно для работы с базой данных(редак., удаление, добавление)
   2) Вызывается кнопкой(троеточием) под таблицей данных
4) products_del.ui
   1) Окно для удаление покупки
   2) Вызывается кнопкой удалить
