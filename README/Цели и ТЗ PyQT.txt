Цель:
Создать приложение, которое будет предназначено для магазинов/супермаркетов для внесение данных о покупках покупателей. Данное приложение поможет работникам/владельцам магазина в простой и удобной работой с кассой, выдачи сдачи/конечной суммы покупки, подведение данных о покупках.

ТЗ:
1) Редактирование(Удаление покупки, редактирование, добавление продукта, редактировать покупку) покупок
2) Суммирование и наличие скидок(забыл указать на картинке)
3) Сумма и вычесление итоговой ценны
4) Ввод данных о покупателе и продавце(дополнительные случаи при не указанние данных)
5) Сохранение чека/покупки
6) Авто ввод данных о покупателе/продавце(снова забыл указать на картинке)
